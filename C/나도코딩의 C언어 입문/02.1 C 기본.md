# C의 기본 구조
```c
#include <stdio.h>

int main(void) {
	printf("Hello World\n");
	return 0;
}
```
## 전처리기 지시문 **\#include**
-  컴파일에 필요한 파일을 코드에 포함
- <파일명> : C 언어에 정의된 표준 파일 추가
- "파일명" : 현재 폴더 내에 존재하는 파일 추가
- stdio.h : 표준입출력 **헤더 파일**
## 함수 **int main(void){ }**
- main 앞의 int : 함수가 **반환**하는 값의 형태 (정수)
- main() 내의 void : 함수가 **입력**받는 값 (없음)
- **return** : 반환하는 값
# 변수
## 변수의 저장
### 예) int age = 20;
- int : 자료형 (정수)
- age : 변수명
- 20 : 변수에 저장할 값
- = : 20을 age에 저장
### 변수명 규칙
- 알파벳, 숫자, _ 가능
- 대소문자 구분
- 첫 글자 숫자 불가, 공백 불가
- C언어에서 키워드로 지정된 단어(int 등) 사용 불가
