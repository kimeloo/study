# 링크
### URL
- 기본 : \[text](url)
	- cmd/ctrl + k
- 참조 : \[text](별칭) 적고, 문서 마지막에 \[별칭]: url
### 메모 링크
- 메모이름(존재하지 않아도 가능) : \[\[file]]
### 특정 부분 링크
- 메모의 섹션 : \[\[filename#section]]
- 메모의 부분 : \[\[filename^linehash]]
### 임베드
- \!\[\[name]]
- 파일(이미지, 오디오) 혹은 메모 임베드
