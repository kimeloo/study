# 디렉터리
### ls
현재 디렉터리 내용 표시
### pwd
현재 디렉터리 경로 표시
### cd
현재 디렉터리 이동
### mkdir
새로운 디렉터리 생성
### rmdir
빈 디렉터리 삭제
### pushd
디렉터리 스택에 추가
### dirs
디렉터리 스택의 내용 표시
### popd
디렉터리 스택 마지막 내용 획득
# 파일
### cp
파일 복사
### rm
파일 삭제
### mv
파일 이동
### ln
파일 링크 생성
### cat
파일 내용 표시
### more
파일 내용을 페이지 단위로 표시
### wc
문서의 줄, 행 등의 정보 표시
### grep
파일 내에서 원하는 문자열, 규칙 찾기
### tar
여러 파일을 하나로 묶기
### compress
파일 압축하기
# 명령어
### man
유틸리티, API, 명령어 정보 표시
### apropos
해당 주제와 관련된 명령어 표시
### which
명령어가 위치한(저장된) 경로 표시
### find
시스템 내에서 명령어 검색
# 권한, 사용자
### sudo
슈퍼 유저 권한으로 실행
### su
사용자 변경
### chmod
파일의 접근권한 변경
### chown
파일의 소유자 변경
### chgrp
파일과 고나련된 그룹 변경
### who
사용자 정보 표시
### wall
모든 사용자에게 메시지 전달
### logout
현재 사용자 로그아웃
### aduser
사용자 추가
### passwd
사용자 비밀번호 변경
# 프로세스
### ps
프로세스의 상태 표시
### kill
프로세스에 시그널 전송
### fg
프로세스를 포그라운드로 전환
### bg
프로세스를 백그라운드로 전환
# 시스템
### top
시스템 상태 표시
### free
현재 남은 메모리 표시
### du
디스크 사용량 검사
### df
디스크 남은 공간 표시
### mount
디바이스를 시스템에 연결
### halt
시스템 정지
### sync
현재 캐시된 내용 저장
### reboot
시스템 재시작
### poweroff
시스템 종료
### startx
X 윈도우 시작
### apt-get
데비안 패키지 설치
### apt-cache
데비안 패키지 검색/표시
# 기타

터미널 화면 지우기
### echo
입력한 문자열 출력
### date
현재 시스템의 날짜 출력
### cal
달력 표시
### telnet
원격 서버에 터미널로 접속
