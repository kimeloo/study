# 링크
### URL
- \[text](url)
- cmd/ctrl + k
### 메모 링크
- 현재 존재하는 메모 : \[\[file]]
- 현재 존재하지 않는 메모 : \[\[no-file]]
### 특정 부분 링크
- 메모의 섹션 : \[\[filename#section]]
- 메모의 부분 : \[\[filename^linehash]]
### 임베드
- \!\[\[name]]
- 파일(이미지, 오디오) 혹은 메모 임베드
